# Tamil Kodi

A Kodi plugin for Tamil trailers, movies, TV Shows. This is a fork of Tamil Kodi plugin.

## Additional Features

- Trailers, HD Movies, News listed in the front page

## Install

Unix-like platforms such as Linux and Mac OS X
```
  ./gradlew
```
Windows
```
  gradlew
```
